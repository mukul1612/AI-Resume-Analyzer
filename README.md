# AI Resume Analyzer 🚀

A full-stack Gen AI job preparation platform that analyzes resumes, matches job descriptions, identifies skill gaps, and generates interview questions.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT00000000000000000000
- AI: Gemini API


## Features
- Resume upload & analysis
- Job description matching
- Skill gap detection
- AI-generated interview questions
- Authentication & authorization
