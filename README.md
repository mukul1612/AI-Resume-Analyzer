# 🚀 AI Resume Analyzer

A full-stack **Gen AI-powered job preparation platform** that analyzes resumes, matches them with job descriptions, identifies skill gaps, and generates personalized interview preparation content.

---

## 🌐 Live Demo
👉 https://ai-resume-analyzer-sepia-two.vercel.app/login  

---

## ⚙️ Tech Stack

- **Frontend:** React (Vite)  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Authentication:** JWT  
- **AI Integration:** Gemini API  

---

## ✨ Features

- 📄 Resume Upload & Analysis  
- 🎯 Job Description Matching  
- 📊 Match Score Calculation  
- 🧠 Skill Gap Identification  
- 🤖 AI-Generated Interview Questions  
  - Technical Questions  
  - Behavioral Questions  
- 📅 Personalized Preparation Plan  
- 📝 AI Resume Generator (based on Job Description)  
- 🔐 Authentication & Authorization (JWT)  

---

## 🧩 How It Works

1. User uploads a **resume** or provides a **self-description**  
2. Inputs a **job description (JD)**  
3. System analyzes and provides:
   - ✅ Match Score  
   - 📉 Skill Gaps  
   - 🎯 Required Skills  
   - 💡 Improvement Suggestions  
4. AI generates:
   - 🧠 Technical Interview Questions  
   - 🤝 Behavioral Questions  
   - 📅 Preparation Plan  
5. User can also:
   - ✨ Generate a **new optimized resume** based on JD
